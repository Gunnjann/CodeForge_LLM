# CodeForge-LLM Web Authoring & Learning System

## Description
CodeForge-LLM is a modern web-based authoring and learning system designed to simplify the process of creating, managing, and learning from web-based projects. It leverages cutting-edge technologies to provide an intuitive and efficient user experience.

## Features
- **Landing Page Components**: Pre-designed sections like Hero, Features, How It Works, and more.
- **Dashboard**: A user-friendly dashboard for managing projects.
- **UI Components**: A library of reusable components such as buttons, forms, tables, and more.
- **Resume Builder**: A tool to create professional resumes.
- **Authentication**: Signup and login pages for user management.
- **Responsive Design**: Optimized for mobile and desktop devices.

## Technologies Used
- **Frontend**: React, TypeScript, TailwindCSS
- **Build Tool**: Vite
- **Testing**: Vitest
- **Linting**: ESLint

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Gunnjann/CodeForge_LLM.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CodeForge-LLM
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project
To start the development server:
```bash
npm run dev
```

### Building for Production
To build the project for production:
```bash
npm run build
```

### Running Tests
To run tests:
```bash
npm run test
```
